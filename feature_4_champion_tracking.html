Champion tracking

Feature allows you to select up to three champions and replay file will be generated if any of them are present in Summoner's game.
Note that as of now feature is not fully implemented due to not being able to run replay file on user's computer. If you can help with this - feel free to contact me by email provided at the bottom of page.

Step 1 - Overview

The purpose of feature is to allow you to save replays when specific champions appear in Summoner's game.
This can be used to review your mistakes, capture mechanics of a good player, save replays of a pro player when he plays champion you like, etc.

Step 2 - Selecting champions for tracking

There are three slots to select a champion for tracking.
To select or change a champion click on any of three buttons in UI below "Tracked Champions" heading.

Step 3 - Navigating in champion list

All champions are sorted by name in ascending order.
You can scroll list by using mouse wheel or scroll bar on the right.
If champion is not present in list - verify that you are using latest version of application.

Step 4 - Selecting champion

To select a champion click on his portrait.

Step 5 - Changing slot state

To change state of tracking slot click on it.
You can either select another champion for tracking or deselect it if you no longer wish to track.

Step 6 - Deselecting or changing champion

In opened window click on gray icon in top left side if you wish to deselect champion.
Or click on another champion to select him instead for tracking.

Step 7 - Closing window without changes

To close window click on red cross button on the center bottom side of window.

Step 8 - Replays

After game ends a replay file will be saved in application directory.
Replay is named with date and tracked champions present in it.
